# Brain

Simple task/notes system managed by Claude Code.

## Usage

Open this directory in Claude Code and use natural language:

```
"Add task: Create fix list for Beacon, tags: BandLab, Beacon"
"List all BandLab tasks"
"Mark task 3 as done"
"What's in progress?"
"Show all tags"
```

## Structure

- `tasks.json` - All tasks with tags and status
- `CLAUDE.md` - Instructions for Claude Code
